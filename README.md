# dauerhaft.ch

Find the stuff that truly lasts.

## why this site rocks
- written in one week
- no binary data (not even for favicon)
- no tracking, single origin
- no javascript (only livewire)
- no DBM (only sqlite)

## Installation

```bash
git clone blemli/dauerhaft
cd dauerhaft
cp .env.example .env
composer install
php artisan make:filament-user
```
